Parcel JS - Using The Docs

By Charles Loehle

Parceljs - Getting Started

  • Install parcel-bundler npm install --save-dev parcel-bundler
  • In package.json add these lines to "scripts":
    • 				
	"scripts": {
		"dev": "parcel src/index.html",
		"build": "parcel build src/index.html"
	},
  • Next, create an index.html and index.js file
  • Bring in index.js file before the ending body tag

SCSS

  • Install sass npm install -D sass
  • Create a .sassrc file and add node_modules as include path to avoid an issue 
    {
	"includePaths": ["node_modules"]
}

To use sass, there are 2 ways of including it

  1. import SCSS files in index.js 
      import './custom.scss'
  2. Or directly include the SCSS file in index.html head.

Spin up the server

	npm run dev

At this point it should be compiling js with sass and hot reloading
